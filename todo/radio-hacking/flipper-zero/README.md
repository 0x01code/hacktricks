Z [**Flipperem Zero**](https://flipperzero.one/) możesz:

* **Słuchać/Zbierać/Odtwarzać częstotliwości radiowe:** [**Sub-GHz**](fz-sub-ghz.md)
* **Czytać/Zbierać/Emulować karty NFC:** [**NFC**](fz-nfc.md)
* **Czytać/Zbierać/Emulować tagi 125kHz:** [**125kHz RFID**](fz-125khz-rfid.md)
* **Czytać/Zbierać/Wysyłać sygnały podczerwieni:** [**Podczerwień**](fz-infrared.md)
* **Czytać/Zbierać/Emulować iButtony:** [**iButton**](../ibutton.md)
* **Używać go jako Bad USB**
* **Używać go jako klucz bezpieczeństwa (U2F)**
* **Grać w Snake**

**Inne zasoby Flipper Zero znajdziesz w** [**https://github.com/djsime1/awesome-flipperzer**](https://github.com/djsime1/awesome-flipperzero)
