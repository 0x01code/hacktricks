* **Ler/Capturar/Enviar sinais infravermelhos:** [**Infravermelho**](fz-infrared.md)****
* **Ler/Capturar/Emular iButtons:** [**iButton**](../ibutton.md)****

* **Usá-lo como chave de segurança (U2F)**
* **Jogar Snake**
