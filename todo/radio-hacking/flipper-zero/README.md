[**Flipper Zero**](https://flipperzero.one/)를 사용하여 다음을 수행할 수 있습니다:

* **라디오 주파수 청취/캡처/재생:** [**Sub-GHz**](fz-sub-ghz.md)****
* **NFC 카드 읽기/캡처/에뮬레이션:** [**NFC**](fz-nfc.md)****
* **125kHz 태그 읽기/캡처/에뮬레이션:** [**125kHz RFID**](fz-125khz-rfid.md)****
* **적외선 신호 읽기/캡처/전송:** [**Infrared**](fz-infrared.md)****
* **iButton 읽기/캡처/에뮬레이션:** [**iButton**](../ibutton.md)****
* **Bad USB로 사용하기**
* **보안 키로 사용하기 (U2F)**
* **스네이크 게임하기**

[**https://github.com/djsime1/awesome-flipperzero**](https://github.com/djsime1/awesome-flipperzero)에서 다른 Flipper Zero 자료를 찾을 수 있습니다.
