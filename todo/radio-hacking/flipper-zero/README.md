* **NFC 카드 읽기/캡처/에뮬레이트:** [**NFC**](fz-nfc.md)

* **적외선 신호 읽기/캡처/전송:** [**Infrared**](fz-infrared.md)
* **iButtons 읽기/캡처/에뮬레이트:** [**iButton**](../ibutton.md)
* **Bad USB로 사용**
* **보안 키(U2F)로 사용**
