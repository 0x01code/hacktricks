Sa [**Flipper Zero**](https://flipperzero.one/) možete:

* **Slušajte/Snimajte/Ponavljajte radio frekvencije:** [**Sub-GHz**](fz-sub-ghz.md)****
* **Čitajte/Snimajte/Emulirajte NFC kartice:** [**NFC**](fz-nfc.md)****
* **Čitajte/Snimajte/Emulirajte 125kHz oznake:** [**125kHz RFID**](fz-125khz-rfid.md)****
* **Čitajte/Snimajte/Šaljite infracrvene signale:** [**Infrared**](fz-infrared.md)****
* **Čitajte/Snimajte/Emulirajte iButtons:** [**iButton**](../ibutton.md)****
* **Koristite ga kao Bad USB**
* **Koristite ga kao sigurnosni ključ (U2F)**
* **Igrajte Snake**

**Ostali Flipper Zero resursi na** [**https://github.com/djsime1/awesome-flipperzero**](https://github.com/djsime1/awesome-flipperzero)****
