## Τιμές του HackTricks

{% hint style="success" %}
Αυτές είναι οι **τιμές του Έργου HackTricks**:

* Δίνουμε **ΔΩΡΕΑΝ** πρόσβαση σε **ΕΚΠΑΙΔΕΥΤΙΚΟ χάκινγκ** πόρους σε **ΟΛΟΥΣ** στο Διαδίκτυο.
* Το χάκινγκ είναι για μάθηση, και η μάθηση πρέπει να είναι τόσο ελεύθερη όσο είναι δυνατόν.
* Ο σκοπός αυτού του βιβλίου είναι να λειτουργήσει ως ένας περιεκτικός **εκπαιδευτικός πόρος**.
* **ΑΠΟΘΗΚΕΥΟΥΜΕ** εκπληκτικές **τεχνικές χάκινγκ** που η κοινότητα δημοσιεύει δίνοντας σε **ΟΛΟΥΣ** τους **ΠΡΩΤΟΤΥΠΟΥΣ ΔΗΜΙΟΥΡΓΟΥΣ** όλα τα **μεγαλεία**.
* **Δεν θέλουμε την αναγνώριση από άλλους ανθρώπους**, θέλουμε απλά να αποθηκεύσουμε ψυχαγωγικά κόλπα για όλους.
* Γράφουμε επίσης **τις δικές μας έρευνες** στο HackTricks.
* Σε πολλές περιπτώσεις θα γράψουμε απλά **στο HackTricks ένα περίληψη των σημαντικών μερών** της τεχνικής και θα **ενθαρρύνουμε τον αναγνώστη να επισκεφθεί την αρχική δημοσίευση** για περισσότερες λεπτομέρειες.
* **ΟΡΓΑΝΩΝΟΥΜΕ** όλες τις τεχνικές χάκινγκ στο βιβλίο έτσι ώστε να είναι **ΠΙΟ ΠΡΟΣΙΤΕΣ**
* Η ομάδα του HackTricks έχει αφιερώσει χιλιάδες ώρες δωρεάν **μόνο για να οργανώσει το περιεχόμενο** έτσι ώστε οι άνθρωποι να **μαθαίνουν πιο γρήγορα**
{% endhint %}

## Συχνές Ερωτήσεις του HackTricks

{% hint style="success" %}
* **Σας ευχαριστώ πολύ για αυτούς τους πόρους, πώς μπορώ να σας ευχαριστήσω;**
{% endhint %}

Μπορείτε να ευχαριστήσετε δημόσια την ομάδα του HackTricks για τη συγκέντρωση όλων αυτών των πόρων δημόσια σε ένα tweet αναφέροντας το [**@hacktricks\_live**](https://twitter.com/hacktricks\_live).\
Αν είστε ιδιαίτερα ευγνώμοντες μπορείτε επίσης να [**υποστηρίξετε το έργο εδώ**](https://github.com/sponsors/carlospolop).\
Και μην ξεχάσετε να **δώσετε ένα αστέρι στα έργα στο Github!** (Βρείτε τους συνδέσμους παρακάτω).

{% hint style="success" %}
* **Πώς μπορώ να συνεισφέρω στο έργο;**
{% endhint %}

Μπορείτε να **μοιραστείτε νέες συμβουλές και κόλπα με την κοινότητα ή να διορθώσετε σφάλματα** που βρίσκετε στα βιβλία στέλνοντας ένα **Αίτημα Ενσωμάτωσης (Pull Request)** στις αντίστοιχες σελίδες του Github:

* [https://github.com/carlospolop/hacktricks](https://github.com/carlospolop/hacktricks)
* [https://github.com/carlospolop/hacktricks-cloud](https://github.com/carlospolop/hacktricks-cloud)

Μην ξεχάσετε να **δώσετε ένα αστέρι στα έργα στο Github!**

{% hint style="success" %}
* **Μπορώ να αντιγράψω κάποιο περιεχόμενο από το HackTricks και να το τοποθετήσω στο blog μου;**
{% endhint %}

Ναι, μπορείτε, αλλά **μην ξεχάσετε να αναφέρετε τους συγκεκριμένους συνδέσμους** από όπου προήλθε το περιεχόμενο.

{% hint style="success" %}
* **Πώς μπορώ να αναφέρω μια σελίδα του HackTricks;**
{% endhint %}

Όσο το σύνδεσμος **της** σελίδας(ων) από όπου πήρατε τις πληροφορίες εμφανίζεται είναι αρκετό.\
Αν χρειάζεστε ένα bibtex μπορείτε να χρησιμοποιήσετε κάτι σαν:
```latex
@misc{hacktricks-bibtexing,
author = {"HackTricks Team" or the Authors name of the specific page/trick},
title = {Title of the Specific Page},
year = {Year of Last Update (check it at the end of the page)},
url = {\url{https://book.hacktricks.xyz/specific-page}},
}
```
{% hint style="warning" %}
* **Μπορώ να αντιγράψω όλα τα HackTricks στο blog μου;**
{% endhint %}

**Θα προτιμούσα όχι**. Αυτό **δεν θα ωφελήσει κανέναν** καθώς όλο το **περιεχόμενο είναι ήδη διαθέσιμο δημόσια** στα επίσημα βιβλία του HackTricks δωρεάν.

Αν φοβάστε ότι θα εξαφανιστεί, απλά κάντε fork στο Github ή κατεβάστε το, όπως είπα είναι ήδη δωρεάν.

{% hint style="warning" %}
* **Γιατί έχετε χορηγούς; Τα βιβλία του HackTricks είναι για εμπορικούς σκοπούς;**
{% endhint %}

Η πρώτη **αξία του HackTricks** είναι να προσφέρει **ΔΩΡΕΑΝ** εκπαιδευτικούς πόρους για το **ΟΛΟΚΛΗΡΟ** τον κόσμο. Η ομάδα του HackTricks έχει **αφιερώσει χιλιάδες ώρες** για να προσφέρει αυτό το περιεχόμενο, ξανά, **ΔΩΡΕΑΝ**.

Αν νομίζετε ότι τα βιβλία του HackTricks είναι φτιαγμένα για **εμπορικούς σκοπούς** είστε **ΕΝΤΕΛΩΣ ΛΑΘΟΣ**.

Έχουμε χορηγούς επειδή, ακόμα κι αν όλο το περιεχόμενο είναι ΔΩΡΕΑΝ, θέλουμε να **προσφέρουμε στην κοινότητα τη δυνατότητα να εκτιμήσει τη δουλειά μας** αν το επιθυμούν. Γι' αυτό, προσφέρουμε στους ανθρώπους τη δυνατότητα να δωρίσουν στο HackTricks μέσω [**Github sponsors**](https://github.com/sponsors/carlospolop), και σε **σχετικές εταιρείες κυβερνοασφάλειας** να χορηγήσουν το HackTricks και να **έχουν κάποιες διαφημίσεις** στο βιβλίο, με τις **διαφημίσεις** να τοποθετούνται πάντα σε σημεία που τις κάνουν **ορατές** αλλά **χωρίς να ενοχλούν τη διαδικασία μάθησης** αν κάποιος επικεντρώνεται στο περιεχόμενο.

Δεν θα βρείτε το HackTricks γεμάτο με ενοχλητικές διαφημίσεις όπως άλλα blogs με πολύ λιγότερο περιεχόμενο από το HackTricks, επειδή το HackTricks δεν είναι φτιαγμένο για εμπορικούς σκοπούς.

{% hint style="danger" %}
* **Τι πρέπει να κάνω αν μια σελίδα του HackTricks βασίζεται στην ανάρτηση του blog μου αλλά δεν αναφέρεται;**
{% endhint %}

**Λυπούμαστε πολύ. Δεν έπρεπε να συμβεί αυτό**. Παρακαλώ, ενημερώστε μας μέσω Github issues, Twitter, Discord... με τον σύνδεσμο της σελίδας του HackTricks με το περιεχόμενο και τον σύνδεσμο του blog σας και **θα το ελέγξουμε και θα το προσθέσουμε ΑΜΕΣΑ**.

{% hint style="danger" %}
* **Τι πρέπει να κάνω αν υπάρχει περιεχόμενο από το blog μου στο HackTricks και δεν θέλω να υπάρχει;**
{% endhint %}

Να σημειωθεί ότι έχοντας συνδέσεις στη σελίδα σας στο HackTricks:

* Βελτιώνει το **SEO** σας
* Το περιεχόμενο μεταφράζεται σε περισσότερες από 15 γλώσσες, κάτι που επιτρέπει σε περισσότερους ανθρώπους να έχουν πρόσβαση σε αυτό το περιεχόμενο
* Το **HackTricks ενθαρρύνει** τους ανθρώπους να **επισκεφτούν τη σελίδα σας** (πολλοί έχουν αναφέρει ότι από τότε που μια σελίδα τους είναι στο HackTricks λαμβάνουν περισσότερες επισκέψεις)

Ωστόσο, αν εξακολουθείτε να θέλετε το περιεχόμενο του blog σας να αφαιρεθεί από το HackTricks απλά ενημερώστε μας και θα αφαιρέσουμε σίγουρα **κάθε σύνδεση προς το blog σας**, και οποιοδήποτε περιεχόμενο που βασίζεται σε αυτό.

{% hint style="danger" %}
* **Τι πρέπει να κάνω αν βρω αντιγραφές περιεχομένου στο HackTricks;**
{% endhint %}

Πάντα **δίνουμε όλα τα εύσημα στους αρχικούς συγγραφείς**. Αν βρείτε μια σελίδα με αντιγραφές περιεχομένου χωρίς αναφορά στην αρχική πηγή, ενημερώστε μας και είτε θα το **αφαιρέσουμε**, θα **προσθέσουμε τον σύνδεσμο πριν το κείμενο**, ή θα το **αναδιατυπώσουμε προσθέτοντας τον σύνδεσμο**.

## ΑΔΕΙΑ

Πνευματικά δικαιώματα © Με επιφύλαξη παντός δικαιώματος εκτός αν αναφέρεται διαφορετικά.

#### Σύνοψη Άδειας:

* Αναφορά: Είστε ελεύθεροι:
* Κοινοποίηση - αντιγραφή και επαναδιανομή του υλικού σε οποιοδήποτε μέσο ή μορφή.
* Προσαρμογή - αναμίξτε, μετασχηματίστε και δημιουργήστε βάσει του υλικού.

#### Επιπλέον Όροι:

* Περιεχόμενο Τρίτων: Ορισμένα τμήματα αυτού του blog/βιβλίου μπορεί να περιλαμβάνουν περιεχόμενο από άλλες πηγές, όπως αποσπάσματα από άλλα blogs ή δημοσιεύσεις. Η χρήση τέτοιου περιεχομένου γίνεται με βάση τις αρχές της δίκαιης χρήσης ή με την ρητή άδεια των αντίστοιχων κατόχων πνευματικών δικαιωμάτων. Παρακαλούμε ανατρέξτε στις αρχικές πηγές για συγκεκριμένες πληροφορίες άδειας σχετικά με το περιεχόμενο τρίτων.
* Συγγραφή: Το αρχικό περιεχόμενο που συντάχθηκε από το HackTricks υπόκειται στους όρους αυτής της άδειας. Σας ενθαρρύνεται να αποδίδετε αυτό το έργο στον συγγραφέα κατά την κοινοποίηση ή προσαρμογή του.

#### Εξαιρέσεις:

* Εμπορική Χρήση: Για ερωτήσεις σχετικά με την εμπορική χρήση αυτού του περιεχομένου, επικοινωνήστε μαζί μου.

Αυτή η άδεια δεν χορηγεί κανένα δικαίωμα εμπορικών σημάτων ή εμπορικών σημάτων σχετικά με το περιεχόμενο. Όλα τα εμπορικά σήματα και σήματα που παρουσιάζονται σε αυτό το blog/βιβλίο είναι ιδιοκτησία των αντίστοιχων κατόχων τους.

**Χρησιμοποιώντας ή έχοντας πρόσβαση στο HackTricks, συμφωνείτε να τηρείτε τους όρους αυτής της άδειας. Αν δεν συμφωνείτε με αυτούς τους όρους, παρακαλώ, μην έχετε πρόσβαση σε αυτό τον ιστότοπο.**
