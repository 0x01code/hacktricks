在进行钓鱼评估时，有时完全**克隆一个网站**可能会很有用。

请注意，您还可以向克隆的网站添加一些有效负载，比如 BeEF 钩子，以“控制”用户的标签页。

有不同的工具可用于此目的：

## wget
```text
wget -mk -nH
```
## 克隆网站
```bash
#https://github.com/imthaghost/goclone
goclone <url>
```
