## ADB communication

ADB рд╕рдВрдЪрд╛рд░ рд╕реЗ рдПрдХ APK рдирд┐рдХрд╛рд▓реЗрдВ рдЬрд╣рд╛рдБ APK рднреЗрдЬрд╛ рдЧрдпрд╛ рдерд╛:
```python
from scapy.all import *

pcap = rdpcap("final2.pcapng")

def rm_data(data):
splitted = data.split(b"DATA")
if len(splitted) == 1:
return data
else:
return splitted[0]+splitted[1][4:]

all_bytes = b""
for pkt in pcap:
if Raw in pkt:
a = pkt[Raw]
if b"WRTE" == bytes(a)[:4]:
all_bytes += rm_data(bytes(a)[24:])
else:
all_bytes += rm_data(bytes(a))
print(all_bytes)

f = open('all_bytes.data', 'w+b')
f.write(all_bytes)
f.close()
```
