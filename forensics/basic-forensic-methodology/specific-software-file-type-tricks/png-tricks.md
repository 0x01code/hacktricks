PNGファイルは、特にCTFチャレンジで人気があります。これは、画像内の非視覚データを隠すのに適した無損失圧縮が理由です。PNGファイルはWiresharkで解析することができます。破損したPNGを確認または修復するためには、[pngcheck](http://libpng.org/pub/png/apps/pngcheck.html)を使用できます。

破損したPNGを修復するために、オンラインツール[https://online.officerecovery.com/pixrecovery/](https://online.officerecovery.com/pixrecovery/)を試すことができます。
