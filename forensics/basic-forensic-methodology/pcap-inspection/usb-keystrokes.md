以下のようなキーボードのUSB経由の通信を含むpcapを持っている場合：

![](<../../../.gitbook/assets/image (613).png>)

[**ctf-usb-keyboard-parser**](https://github.com/carlospolop-forks/ctf-usb-keyboard-parser)ツールを使用して、通信で何が書かれていたかを取得できます：
```bash
tshark -r ./usb.pcap -Y 'usb.capdata && usb.data_len == 8' -T fields -e usb.capdata | sed 's/../:&/g2' > keystrokes.txt
python3 usbkeyboard.py ./keystrokes.txt
```
以下のリンクには、この分析方法についての詳細情報とスクリプトがいくつかあります：
