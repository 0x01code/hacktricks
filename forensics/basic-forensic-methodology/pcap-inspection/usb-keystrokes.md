Si tienes un pcap que contiene la comunicación vía USB de un teclado como el siguiente:

![](<../../../.gitbook/assets/image (613).png>)

Puedes usar la herramienta [**ctf-usb-keyboard-parser**](https://github.com/carlospolop-forks/ctf-usb-keyboard-parser) para obtener lo que se escribió en la comunicación:
```bash
tshark -r ./usb.pcap -Y 'usb.capdata && usb.data_len == 8' -T fields -e usb.capdata | sed 's/../:&/g2' > keystrokes.txt
python3 usbkeyboard.py ./keystrokes.txt
```
Puede leer más información y encontrar algunos scripts sobre cómo analizar esto en:
