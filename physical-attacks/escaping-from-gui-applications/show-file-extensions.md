## 如何在Windows XP中显示文件扩展名

1. 打开Windows资源管理器。您可以通过从开始菜单点击“我的电脑”来做到这一点。注意，这与Internet Explorer不是同一回事。
2. 在资源管理器的菜单栏（菜单栏位于资源管理器窗口的顶部，窗口标题下方）点击“工具”菜单。在弹出的菜单中点击“文件夹选项”。
3. 将出现一个对话框。点击窗口顶部的“查看”标签。
4. 寻找“隐藏已知文件类型的扩展名”设置，并取消选中旁边的复选框以禁用它。
5. 点击“对所有文件夹应用”按钮。然后点击“应用”和“确定”。

## 在Windows Vista和Windows 7中显示文件扩展名

1. 点击开始菜单。这是屏幕左下角的圆形图标。
2. 输入“文件夹选项”（不带引号）。点击开始菜单顶部出现的“文件夹选项”行。
3. 将出现一个标题为“文件夹选项”的对话框。点击窗口顶部的“查看”标签。
4. 取消选中“隐藏已知文件类型的扩展名”的复选框。
5. 点击对话框底部的“确定”按钮。

## 如何在Windows 8中显示文件扩展名

1. 调出开始屏幕。一种方法是将鼠标移动到屏幕的左下角。在Windows 8中，没有迹象表明这会导致开始屏幕出现，但它会出现。
2. 输入“文件夹选项”（不带引号）。我知道开始屏幕上没有空白字段来提示您实际上可以输入任何内容。像Windows 8中的许多事情一样，需要神秘的知识来操作系统。
3. 屏幕左侧，您将得到令人失望的消息“没有应用与您的搜索匹配”。Windows 8的开始屏幕搜索功能只有在您已经知道在哪里找到它并且可以告诉它它在哪里的情况下才能为您找到东西。

点击屏幕右侧的“设置”行。
4. 屏幕左侧现在显示“文件夹选项”作为可能的结果之一。点击它。
5. 将出现一个标题为“文件夹选项”的窗口。点击该对话框顶部的“查看”标签。
6. 寻找“隐藏已知文件类型的扩展名”一行，并点击旁边的框以去除勾选。
7. 点击“确定”按钮。

## 如何在Windows 10中显示文件扩展名

1. 点击屏幕底部任务栏上的图标以调出开始菜单。（图标是Windows徽标的白色版本。）
2. 输入“文件夹选项”（不带引号）。屏幕上没有空白字段来提示您可以输入任何内容，但您可以。只需输入即可。
3. 点击出现在“最佳匹配”列表中的“文件资源管理器选项”项。
4. 将出现一个标题为“文件资源管理器选项”的对话框。点击窗口顶部的“查看”标签。
5. 滚动找到“隐藏已知文件类型的扩展名”项，并点击旁边的框以去除勾选。
6. 点击“确定”按钮。

就是这样。您现在应该能够在资源管理器窗口中看到文件的真实扩展名。
