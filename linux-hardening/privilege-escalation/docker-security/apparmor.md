**Resumen del perfil docker-default**:

- **Acceso** a toda la **red**
- No se define ninguna **capacidad** (Sin embargo, algunas capacidades vendr√°n al incluir reglas base b√°sicas, es decir, #include \<abstractions/base>)

Una vez que **ejecutes un contenedor docker**, deber√≠as ver la siguiente salida:

Ten en cuenta que **apparmor incluso bloquear√° los privilegios de capacidades** concedidos al contenedor de forma predeterminada. Por ejemplo, ser√° capaz de **bloquear el permiso de escritura dentro de /proc incluso si se concede la capacidad SYS\_ADMIN** porque por defecto el perfil de apparmor de docker niega este acceso:
