## PETOOH

PETOOH é uma linguagem esotérica que usa a sintaxe de uma máquina de turing. A linguagem é baseada em um conjunto de instruções que são representadas por palavras em russo. Cada palavra tem um valor numérico associado a ela e é usada para manipular um ponteiro de memória. O ponteiro de memória é usado para armazenar e recuperar valores. A linguagem é conhecida por ser difícil de ler e escrever, mas é uma das linguagens esotéricas mais populares.
```
KoKoKoKoKoKoKoKoKoKo Kud-Kudah
KoKoKoKoKoKoKoKo kudah kO kud-Kudah Kukarek kudah
KoKoKo Kud-Kudah
kOkOkOkO kudah kO kud-Kudah Ko Kukarek kudah
KoKoKoKo Kud-Kudah KoKoKoKo kudah kO kud-Kudah kO Kukarek
kOkOkOkOkO Kukarek Kukarek kOkOkOkOkOkOkO
Kukarek
```
