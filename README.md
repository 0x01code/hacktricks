<figure><img src=".gitbook/assets/stm (1).png" alt=""><figcaption></figcaption></figure>

### [Intruder](https://www.intruder.io/?utm\_source=referral\&utm\_campaign=hacktricks)

<figure><img src=".gitbook/assets/image (675).png" alt=""><figcaption></figcaption></figure>

사이버 보안 게임에서 한 발 앞서기세요.

[**Intruder**](https://www.intruder.io/?utm\_source=referral\&utm\_campaign=hacktricks)는 취약성 관리를 쉽게 만들어줍니다. 공격 대상을 추적하고 회사의 취약성을 확인하며 시스템을 가장 노출시키는 문제에 우선순위를 부여하여 중요한 사항에 집중할 수 있도록 도와줍니다.

내부 인프라에서 웹 앱, API 및 클라우드 시스템까지 전체 기술 스택을 포괄하는 단일 플랫폼에서 수천 개의 체크를 실행하세요. [AWS, GCP, Azure](https://www.intruder.io/cloud-vulnerability-scanning-for-aws-google-cloud-and-azure)와 원활하게 통합하여 DevOps를 간소화하여 팀이 수정 사항을 더 빠르게 구현할 수 있도록 합니다.

Intruder는 절대 쉬지 않습니다. 24/7 시스템을 모니터링하여 보호합니다. 더 알고 싶으신가요? 사이트를 방문하여 [**무료 평가판**](https://www.intruder.io/?utm\_source=referral\&utm\_campaign=hacktricks)을 이용해보세요.

{% embed url="https://www.intruder.io/?utm_campaign=hacktricks&utm_source=referral" %}

### [HACKENPROOF](https://bit.ly/3xrrDrL)

<figure><img src=".gitbook/assets/image (5) (1).png" alt=""><figcaption></figcaption></figure>

경험 많은 해커와 버그 바운티 헌터와 소통하려면 [**HackenProof Discord**](https://discord.com/invite/N3FrSbmwdy) 서버에 참여하세요!

**해킹 인사이트**\
해킹의 스릴과 도전을 다루는 콘텐츠와 상호 작용하세요.

**실시간 해킹 뉴스**\
실시간 뉴스와 통찰력을 통해 빠르게 변화하는 해킹 세계를 따라가세요.

**최신 공지사항**\
새로운 버그 바운티 출시 및 중요한 플랫폼 업데이트에 대한 정보를 받아보세요.

**[Discord](https://discord.com/invite/N3FrSbmwdy)**에 참여하여 최고의 해커들과 협업을 시작하세요!

### [Pentest-Tools.com](https://pentest-tools.com/) - 필수적인 펜트스팅 툴킷

<figure><img src=".gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

**취약성 평가 및 펜트스팅을 위한 즉시 사용 가능한 설정**. 20개 이상의 도구 및 기능으로 전체 펜트스팅을 어디서나 실행하세요. 우리는 펜테스터를 대체하지 않습니다 - 그들에게 더 깊이 파고들고, 쉘을 팝하고, 재미있는 시간을 보낼 수 있도록 사용자 정의 도구, 탐지 및 공격 모듈을 개발합니다.
